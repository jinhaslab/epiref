작업 정보: kosha2011_026_027

구분 내용
생산품 LCD모듈 (LCD 패널, 구동회로, 백라이트)
작업도구 용접기, 망치, 포크레인, 컨베이어 벨트, 혼합기, 지대백, 아세톤, IPA, flux, 납, IVIG, 베타페론
작업방식 LCD모듈 조립 공정: 회로연결부부착(OLB) → PCB부착(TAB solder) → 백라이트 조립 → 섀시조립. OLB 공정에서는 불량인 판넬을 수동으로 지우기 위해 아세톤으로 닦고 IPA로 닦기. TAB solder 공정에서는 불량이 생긴 판넬을 수동으로 납땜. 납땜 관련 작업 환경 측정 후 연장근무
작업공간 지상, 지하, 밀폐공간, 환기 불량, 분진 발생 (납 농도 측정)
