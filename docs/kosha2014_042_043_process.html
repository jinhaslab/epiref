구분 내용
생산품 알루미늄 표면처리 제품
작업도구 크레인, 탱크, 약품 처리 시설, 국소 배기시설, 공기 시료 채취 장비
작업방식 알루미늄 제품 입고 → 레크 → 탈지 → 에칭 → 중화 → 전해 피막 → 전착도장 → 건조 → 해체 → 포장
작업공간 반 폐쇄된 피막 작업장, 내부에 무기산, 도료 등 여러 화학물질이 담긴 개방된 탱크가 연속적으로 나열되어 있는 공간. 별도의 출입문 없이 입구가 개방되어 있고 각 약품 처리 탱크에는 측방흡인형 국소 배기시설이 설치되어 작동하나 점검이 미흡하며 강한 약품 냄새가 풍부함. 공기 시료 채취 장비를 이용한 벤젠, 포름알데히드 측정 결과 기반으로 벤젠 연간 누적 노출량은 0.53ppm 추정.
