작업 정보: kosha2014_020_021

구분 내용
생산품 PCB (Printed Circuit Board)
작업도구 수세기, AGV (무인반송대차), 집진기, 국소배기장치, solder paste, 세척제, 방진마스크, 방진장갑, 방진복, 방진모자
작업방식 1. 라우터 공정: 기판을 수세기를 통해 범프반으로 넘겨주기 2. 범프공정: AGV를 이용하여 싱글레이션 공정으로 기판을 넘겨주기 3. 싱글레이션 공정: 습식 환경에서 분진 발생 방지를 위해 국소배기장치 설치 4. 작업 시 방진마스크, 방진장갑, 방진복, 방진모자 착용
작업공간 양압 유지, 격벽 분리된 공간, 습식 환경
